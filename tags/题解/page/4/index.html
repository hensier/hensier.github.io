P1348 Couple number 题解

【题目理解】我们可以很容易地注意到，该题目要求我们在[n1,n2]区间找到满足题意的Couple number $N$。

【第一步：题目转换】
由题意可知，$N=a^2-b^2(N∈[n1,n2]$且$a,b∈Z)$。
而$a^2-b^2=(a+b)(a-b)$。

【第二步：等量代换，换元求解】
假设$a-b=k$，则$a=b+k$。即$k(k+2b)=N$。我们可以注意到，$k$和$k+2b$奇偶性必定相同。

P6056 [加油武汉]SIR 模型 题解

本蒟蒻第一次发表题解，请大家多多包涵。

我们首先将几个变量输入。一开始恢复者人数为$0$，因此总人数$p$保持为$S[0]+I[0]$。每一天，$S$、$I$和$R$的值都会不断发生改变。我们可以在每一次循环（每一天）里新建两个变量来控制$I$和$R$值的变化。这时可以用$cmath$函数来上取整。

题目中的细节是“若计算被感染人数超过易感者人数则全员被感染”。而$S$的值是通过$I$和$R$的值来决定的（$S=p-I-R$），但$I+R$可能会大于$p$。此时$S<0$。人数不可能为负，因此要令$S=0$，并让$I$和$R$的值随之变化，使其满足$p=I+R$（因为这个时候$S=0$）。
